# tetris-3d
3d tetris

# Controls
* left / right arrow keys: move left and right
* up: rotate
* down: speed up
* space: place piece
* 1: reset view
* 2-6: change view angle
* z: toggle camera follow piece
* r: toggle rotation around board

# Building

Depends on sdl2

```
git clone https://github.com/longwatermelon/tetris-3d
cd tetris-3d
make
./tetris
```
